## Kurulum ve Çalıştırma
1. Projeyi klonlayın:
   git clone https://github.com/tugceyildizz2/mini_katalog/new/main?filename=README.md
2. Proje klasörüne girin:
   cd mini_katalog

3. Paketleri yükleyin:
   flutter pub get

4. Uygulamayı çalıştırın:
   flutter run
